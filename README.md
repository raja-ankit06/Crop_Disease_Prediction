An intelligent web-based application that helps farmers and agricultural experts detect crop diseases using image-based machine learning models. The system analyzes leaf images and predicts possible diseases, enabling early diagnosis and improving crop yield.

Features:

📸 Upload crop/leaf images for analysis
🤖 AI-powered disease prediction using trained ML models
📊 Accurate classification of multiple crop diseases
🌐 User-friendly web interface
⚡ Real-time prediction results
📱 Responsive design (works on mobile & desktop)
